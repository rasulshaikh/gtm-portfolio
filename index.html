Workflow

Seven phases, one loop

Each phase maps to a repo or production tool. Gates sit between generation and send so bad data never reaches a prospect.

  1. 01

    Capture signal

    LinkedIn engagers, funding, hiring, tech changes, content drafts

    PhantomBuster, HeyReach, Trigify, founder-led-loop
  2. 02

    Enrich

    Firmographics, technographics, contact data, corpus for content gate

    Apollo, Clay, Prospeo, Deepline
  3. 03

    Score and route

    0-100 ICP score to Hot / Warm / Nurture tiers

    gtm-ai-lead-scorer, Clay formulas
  4. 04

    Assemble copy

    Signal-led openers, hash-seeded variants, cadence hooks, citation proof

    Omnibound cols 52-59, cadences, Clay IIFE
  5. 05

    Gate before send

    List grade B+, col 60 PASS, content gate PASS/FLAG (outbound and CMS)

    list-quality-scorecard, neevcloud-content-gate
  6. 06

    Send

    Email sequences and LinkedIn DMs by tier via n8n

    Smartlead, HeyReach, HubSpot
  7. 07

    Mine and loop

    Calls to content angles to next signal cycle

    founder-led-loop, deliverability audit
Why seven phases: Copy and list QA used to happen in Slack threads. Now they are code: col 60 for outbound, content gate for CMS, list scorecard for uploads. Failures route back with reasons attached.

Copy Framework

Campaign copy, confirmed in four steps

From the campaign-copywriting workflow: direction, opener strategy, body structure, then final variants. No batch copy without sign-off at each step.

1

Direction

Audience, pain, value prop, proof point, AI variables available

2

Opener strategy

Pick Problem Sniffing, Billboard, or Signal-Led for subject + first line

3

Body structure

Value prop placement, case study reference, CTA style, follow-up arc

4

Final copy

All variants, follow-ups, Clay/Smartlead field mapping

First-line strategies in production

Problem Sniffing

When audit data or rankings exist

I asked ChatGPT which [category] vendors to use. You ranked 15th behind [competitor].

Used in: Omnibound citation columns (14-26)

Billboard

When offer is clear, data is thin

We help customers reach their entire TAM every two months.

Used in: RemoteState VP/CTO cadences

Signal-Led

When recent news, hire, or funding exists

Post-raise, pipeline velocity matters. Part of that now starts with AI search.

Used in: Omnibound v14 (replaced "Saw the headline")

OKRs

Q2 2026 GTM Engineering targets

Three credible OKR sets. North star: positive reply rate on signal-led outbound. Pick one set per quarter, track KRs weekly.

Set A - Quality Gates

Nothing ships without passing a gate

  • 100% of lists grade B+ before Smartlead upload
  • Col 60 validator PASS rate >= 95%
  • Content gate blocks near-duplicates (similarity >= 0.55)
Set B - Deploy Velocity

Campaign live in 3 days, not 2 weeks

  • ICP sign-off to live campaign in <= 3 business days
  • 144-variant copy in Clay, zero manual paste
  • n8n push failure rate < 2%
Set C - Compounding Loop

Replies and calls feed the next signal

  • 80% of Hot-tier engagers contacted within 24h
  • 4 founder posts/month from call mining
  • Positive reply rate >= 2% on signal-led campaigns
